# 👟 Shoes Website

This is a responsive e-commerce shoe website built using **HTML**, **CSS**, and **JavaScript**. It features dynamic functionalities such as a shopping cart, product liking, and interactive UI effects.

## 🌐 Live Demo
[Click here to visit the website](https://priya-sharma06.github.io/Shoes-website/)

---

## 📸 Screenshots

### 🏠 Home Page
Displays hero section with Nike promotion and navigation bar.

### 🛍️ Products Page
Showcases Nike shoes with features:
- Product name, price, and star rating
- “Add to Cart” button
- “Like” and “Share” icons
- Stylish product cards

### ℹ️ About Section
Displays product images and description styled with modern design.

### ⭐ Reviews
Customer testimonials with names, photos, and ratings.

### 🛠 Services
Features icons and descriptions for:
- Fast Delivery
- 10 Days Replacement
- 24x7 Support

### 🧾 Cart
Popup showing added products with prices and remove functionality.

### ❤️ Liked Products
Popup listing products marked as liked with remove functionality.

### 🔐 Login Page
Basic login form with social login icons (Google, Facebook, Twitter).

---

## 🚀 Features

- Add to Cart functionality

- Responsive layout
- Interactive buttons and effects
- Scrollable sections with smooth transitions
